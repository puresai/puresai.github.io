这个demo简单使用了组件引入，及子组件于父组件通信。

引入子组件

import TestPage from '@/views/components/TestPage'

export default {
  components: { TestPage },

使用:

<test-page></test-page>

父组件传递数据到子组件

父组件的status就是传递的数据

<test-page status="0"></test-page>

子组件接收：

this.$props.status

子组件传递数据到父组件

子组件，其实是传递了一个toP事件

this.$emit('toP', this.listQuery)

父组件

<test-page @toP="funct"></test-page>

...

methods: {
    funct(v) {
        console.warn(v)
    }
}
