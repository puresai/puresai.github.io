解决了？

前往kibana的Index Management界面，依旧存在 42 indices have lifecycle errors。

手动来一下：

POST /sai-log-2021-11-13/_ilm/retry

output:
{
  "acknowledged" : true
}

重复explain，发现输出一致，再去kibana的Index Management界面瞅瞅：

还有一个，retry命令处理下即可。

基本算是解决了。
